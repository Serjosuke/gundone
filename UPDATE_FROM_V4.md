# Обновление Atlas Forge с v0.4 до v0.5

1. Остановите проект:

```powershell
docker compose down
```

2. Сохраните из старой папки:
   - `.env`
   - `backend/data/` — в ней база SQLite, фоновые изображения и все загруженные файлы.

3. Распакуйте v0.5 и верните эти файлы в новую папку.

4. Пересоберите контейнеры:

```powershell
docker compose up --build
```

## Что сохранится

Все статьи, таймлайн, старая карта, её метки, регионы и изображения сохраняются. После запуска к ним добавляется возможность создавать дополнительные карты.

## Важно

Статьи являются общими для всех карт. При удалении карты удаляются только её локальные метки, регионы и изображения. Сами статьи, их связи и события таймлайна остаются в библиотеке.
