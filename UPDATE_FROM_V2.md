# Обновление с v0.2 на v0.3

1. Остановите старый проект:

```powershell
docker compose down
```

2. Замените файлы проекта файлами из этой версии, **но сохраните папку** `backend/data`, если в ней уже есть ваши статьи и изображения.

3. Проверьте `.env`: `ADMIN_PASSWORD` и `AUTH_SECRET` должны остаться вашими.

4. Соберите и запустите:

```powershell
docker compose up --build
```

База обновится сама при первом старте: будут добавлены поля для карточек-меток на карте. Ручные SQL-команды не нужны.

## Новое управление в редакторе

- **Карта** — двигать карту ЛКМ.
- **Метка** — клик по пустому месту создаёт новую статью с меткой; метки можно тянуть ЛКМ.
- **Граница** — ставьте точки, затем сохраните контур. Нажмите на существующую область, чтобы изменить её, удалить или перерисовать.
- **Объекты** — ЛКМ перетаскивает изображения. ПКМ не используется, контекстное меню на карте отключено.
- **+ Карточка** — загрузите картинку, после чего выберите статью в редакторе объекта. На публичной карте она будет открывать эту статью.
